MySql Commands

Command Description
cd /usr/local/mysql/bin/ "opens up" MySql
./mysql -u root -p Connecting to MySql
Create database if not exists 'database'; Creates a new database
Show databases; Allows you to view all created databases
Use 'database'; Changes the database you are working on
Show tables; Shows you all tables within the selected database
Select 'table'; Selects/opens up the table you want to work on
Select * from 'table'; Shows the contents of the table
Select 'column' from 'table'; Shows you specific columns from the selected table
