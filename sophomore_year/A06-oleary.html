My MySQL Reference Page

Database Commands

Syntax Description
cd /usr/local/mysql/bin/ "opens up" MySql
./mysql -u root -p Connecting to MySql
Create database if not exists 'database'; Creates a new database
Show databases; Allows you to view all created databases
Use 'database'; Changes the database you are working on

Table Column Commands

Syntax Description
Select * from 'database'; Displays all tables in that database
Create table if not exists 'table'; Creates a new table
Create table 'table' values(fname Char(20), age INT); Creates a table with values embedded in it
Explain 'tabe'; Shows the values that the table has (Ex: fname, Char(20))
Select * from 'table'; Shows the contents that is contained in the table

Column Attributes and Modifiers

Syntax Description
Int An integer -2,147,483,648 to 2,147,483,647
DECIMAL A floating point number tha can specify the number of permissible digits. For example, Decimal(5,2) permits -999.99 to 999.99
DOUBLE A double-percision floating point number
DATE A date in teh YYY-MM-DD format
TIME A time in the HH:MM:SS format
DATETIME A combined date and time in the format YYYY-MM-DD HH:MM:SS
TIMESTAMP Automatic date and time of last record entry
CHAR() A string of defined fixed length up to 22 characters long. For example CHAR(100) pads a smaller string to make it 100 characters long
VARCHAR() A string of variable length up to 255 characers long that is stored without padding
TEXT A strin gup to 65,535 characters long
BLOB A binary type for variable data
ENUM() A single string value from a defined list. For example, ENUM("red", "green", "blue") allows entry of any one of these three colors only
SET() A string gor mulitple strings form a defined list. For example, SET("red", "green", "blue") allows entry of any one, or more, of these colors

Table Data Commands

Syntax Description
alter table 'table' add column 'column' datatype; This command allows you to add another column to the table
alter table 'table' drop column 'column'; This command "delets" the specified column
alter table 'table' modify column 'column' datatype; this command allows you to change the datatype of a column
update 'table' set 'column' = 'value' where 'condition'; This command allows you to update a row in the table
